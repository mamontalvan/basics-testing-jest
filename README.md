### Introduction to Testing with JEST:

## 1. Install development dependences:

```
npm i -D typescript jest ts-jest @types/jest ts-node
```

## 2. Configure file for JEST

Option 1:
```
npx ts-jest config:init
```

Option 2:
```
npx jest --init
```
